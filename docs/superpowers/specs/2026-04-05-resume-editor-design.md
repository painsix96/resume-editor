# 网页简历编辑软件设计文档

## 1. 项目概述

创建一个基于React的渐进式Web应用（PWA），为职场人士提供专业、易用的简历编辑工具。通过拖拽式界面和实时预览功能，用户可以轻松创建、编辑和导出专业简历。

### 1.1 目标用户
- 职场人士：需要更新简历，突出工作经验和专业技能

### 1.2 核心价值
- 简化简历创建过程
- 提供专业模板和编辑工具
- 支持智能简历压缩
- 实现本地数据存储
- 支持PDF导出

## 2. 技术架构

### 2.1 技术栈
- **前端框架**：React
- **状态管理**：React Context API + useReducer
- **UI组件**：自定义组件 + CSS Modules
- **拖拽功能**：react-dnd
- **富文本编辑**：slate（选择理由：slate提供更现代的API和更好的扩展性，适合复杂的富文本编辑需求）
- **PDF导出**：html2pdf.js
- **本地存储**：localStorage
- **PWA支持**：Workbox

### 2.2 项目结构
```
/src
  /components
    /common          # 通用组件
    /editor          # 编辑器组件
    /preview         # 预览组件
    /templates       # 模板组件
  /context           # 状态管理
  /hooks             # 自定义钩子
  /utils             # 工具函数
  /assets            # 静态资源
  /styles            # 全局样式
  App.jsx            # 应用主组件
  index.jsx          # 应用入口
  service-worker.js  # PWA服务工作线程
```

## 3. 核心功能设计

### 3.1 模板选择
- 提供5-10个基础专业模板
- 支持实时预览模板效果
- 模板分类：通用、IT、金融、营销等

### 3.2 拖拽编辑
- 支持模块拖拽排序（个人信息、教育背景、工作经验、技能等）
- 支持添加/删除模块
- 拖拽操作实时反馈

### 3.3 富文本编辑
- 支持文本格式化（粗体、斜体、下划线等）
- 支持列表（有序/无序）
- 支持链接插入
- 支持项目符号和编号

### 3.4 实时预览
- 编辑时实时更新预览效果
- 支持切换不同设备预览（桌面、移动设备）
- 预览区域可调整大小

### 3.5 智能简历压缩
- **压缩策略**：通过调整行间距、字号大小、段落间距等方式将简历压缩至一页
- **压缩算法**：
  1. 分析简历内容长度和当前布局
  2. 计算目标压缩比例
  3. 按优先级调整：首先调整行间距，然后调整字号，最后调整段落间距
  4. 实时计算页面占用，确保内容不溢出
- **边界处理**：当内容过多无法压缩时，提供友好提示，建议用户精简内容
- **保持原始内容**：压缩过程中保持用户输入的原始内容不变，仅调整样式属性

### 3.6 本地存储
- 使用localStorage存储用户编辑的简历数据
- 自动保存功能，防止数据丢失
- 支持多个简历版本管理
- 数据导出/导入功能：允许用户将简历数据导出为JSON文件，以便备份和迁移
- 考虑localStorage容量限制（约5MB），实现数据压缩和清理策略

### 3.7 PDF导出
- 支持导出为PDF格式
- 导出前预览最终效果
- 优化PDF布局和格式

### 3.8 多语言支持
- 支持中英文界面切换
- 支持中英文简历内容

## 4. 用户界面设计

### 4.1 整体布局
- 顶部导航栏：品牌标识、功能菜单、语言切换
- 左侧边栏：模板选择、模块管理
- 中央编辑区：富文本编辑器、拖拽操作区
- 右侧预览区：实时简历预览
- 底部工具栏：保存、导出、智能压缩等功能

### 4.2 界面风格
- 简约现代风格
- 配色方案：主色调 #4A90E2，辅助色 #F5F7FA，文字色 #333333
- 字体：系统默认无衬线字体，标题使用稍大字号和加粗效果
- **响应式设计**：
  - **桌面端**（≥1200px）：三栏布局，左侧模板选择，中央编辑区，右侧预览区
  - **平板端**（768px-1199px）：两栏布局，左侧模板选择和编辑区，右侧预览区
  - **移动端**（<768px）：单栏布局，通过标签页切换模板选择、编辑和预览功能
  - **触控优化**：增大移动端点击区域，优化拖拽操作的触摸体验

### 4.3 交互设计
- 拖拽操作：流畅的拖拽动画和反馈
- 表单填写：实时验证和提示
- 按钮状态：明确的hover、active状态
- 加载状态：适当的加载动画
- 错误提示：友好的错误信息和解决方案

## 5. 数据结构设计

### 5.1 简历数据结构
```javascript
{
  id: String,           // 简历唯一标识
  title: String,        // 简历标题
  templateId: String,   // 模板ID
  language: String,     // 语言（zh/en）
  sections: [           // 简历模块
    {
      id: String,       // 模块ID
      type: String,     // 模块类型（personal, education, experience, skills等）
      title: String,    // 模块标题
      content: Object,  // 模块内容（根据类型不同结构不同）
      order: Number     // 模块顺序
    }
  ],
  createdAt: Date,      // 创建时间
  updatedAt: Date       // 更新时间
}
```

### 5.2 模板数据结构
```javascript
{
  id: String,           // 模板ID
  name: String,         // 模板名称
  category: String,     // 模板分类
  preview: String,      // 预览图片URL
  styles: Object,       // 模板样式定义
  defaultSections: Array // 默认模块结构
}
```
